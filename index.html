About Me

Hey — I'm Errald, a Grade 11 student from the Philippines with a serious passion for building things on the web. What started as curiosity turned into a full obsession: in less than a year, I went from zero to building full-stack web experiences with real users.

Before code, I was already deep in the creative world — doing freelance video editing, film production, layout design, and graphic work. That background shaped how I approach web design: everything I build has to feel intentional, atmospheric, and alive.

My biggest project so far is Emerson Memory Palace — a full class yearbook website with Firebase, Cloudinary, password-protected edit modes, real-time sync, and a custom UI I designed from scratch. Built it for my section. They love it.
